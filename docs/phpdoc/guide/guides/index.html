Getting started

Is this the first time you work with phpDocumentor? You have come to the right section in the documentation! In the chapters included of this section we will talk you through the installation, configuration and how to generate your documentation.

  • Note: The folder name 'assets' is the default value provided. It can be override on the 'gulpfile.js'.
  • If no config files are present in the project, the default theme can be specified in a parameter the parameter in the task call:
  • docker-compose vendor/bin/run 46583550e89b4cecd42d1706bd4b0e2e8c001019 --default-theme=your-theme
